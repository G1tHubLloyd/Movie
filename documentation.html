GET /movies

Returns a list of all movies.

Method: GET

Response:

[
  {
    "title": "Inception",
    "genre": "Sci-Fi",
    "director": "Christopher Nolan",
    "releaseYear": 2010,
    "rating": 8.8,
    "isFeatured": true
  }
]

GET /movies/:title

Returns details about a single movie by title.

Method: GET

Example: /movies/Inception

GET /genres/:genre

Returns description and examples of movies in a genre.

Method: GET

Example: /genres/Sci-Fi

GET /directors/:name

Returns information and movies by a specific director.

Method: GET

Example: /directors/Christopher%20Nolan

POST /movies

Adds a new movie to the database.

Method: POST

Body:

{
  "title": "Interstellar",
  "genre": "Sci-Fi",
  "director": "Christopher Nolan",
  "releaseYear": 2014,
  "rating": 8.6
}

POST /users

Registers a new user.

Method: POST

Body:

{
  "username": "janedoe",
  "email": "jane@example.com",
  "password": "secure123",
  "dateOfBirth": "1995-06-15"
}

GET /users

Returns a list of all registered users.

Method: GET

PUT /users/:id

Updates user information.

Method: PUT

Body:

{
  "email": "newemail@example.com",
  "password": "newpass456"
}

POST /users/:id/favorites/:movieId

Adds a movie to a user's favorites.

Method: POST

DELETE /users/:id/favorites/:movieId

Removes a movie from a user's favorites.

Method: DELETE

DELETE /users/:id

Deregisters a user.

Method: DELETE
